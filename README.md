## 4-Bit-Adder-Subtractor-circuit
## Overview
This project demonstrates the design and working of a 4-bit adder-subtractor circuit used to perform binary arithmetic operations. The circuit uses full adders and XOR gates to execute both addition and subtraction using the 2’s complement method.

## Objective
To design and understand the working of a 4-bit adder-subtractor using logic gates and full adders.

## Components Used
- XOR Gates
- Full Adders
- Switches
- LEDs
- Breadboard / Simulation Software

## Working Principle
- When control input M = 0:
  - Circuit performs Addition
- When control input M = 1:
  - Circuit performs Subtraction using 2's complement method
